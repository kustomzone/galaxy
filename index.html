Galaxy is the modern front-end framework based on conversational language.

Galaxy is friendly for new and experienced designers alike

Galaxy isn't just any framework; it's the only one fully built upon human, conversational language. We think the way you'd describe a button to a friend is exactly the way you'd write it in your code.

Write Markup, not Css

A framework shouldn't be something that you need to configure. It should work right out the box. You, as the designer, should spend less time coding and more effort building.

Preprocessors built in

Although the package contains Css, Galaxy was built with Sass, meaning the entire framework is both easily configurable and comprehensable.

Modern responsiveness

Responsiveness is built in, which means your app will work on any mobile, tablet or desktop size. Not only that, each component is carefully crafted so it feels responsive.

This is all possible by rethinking Html

Css was not designed for elegance. When put together with Html, it creates a hassle for the designer that we believe is a much larger problem.
